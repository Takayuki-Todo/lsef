# lsef

lsef (List Extended Features) is an extended version of the `ls` command.

## Overview

`lsef` is a command-line tool written in Rust that extends the standard `ls` command with additional features.

## Requirements

- [Rust](https://www.rust-lang.org/) (edition 2024 or later)

## Build

```bash
cargo build --release
```

## Usage

```bash
./target/release/lsef
```

## Development

Run in development mode:

```bash
cargo run
```

Run tests:

```bash
cargo test
```
